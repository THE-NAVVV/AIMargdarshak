# AIMargdarshak

A modern AI-powered travel planning website built as part of a selection interview task.

## About The Project

This project was created to clone and recreate a modern travel planning platform UI with smooth animations, responsive design, and interactive sections.

The website focuses on delivering a premium travel-planning experience using HTML, CSS, and JavaScript.

## Features

- Modern responsive UI
- Smooth scrolling animations
- AI travel wizard interface
- Destination slideshow
- Saved itineraries overlay
- Team section overlay
- Interactive navigation effects
- Clean glassmorphism-inspired design
- Fully responsive layout

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Purpose

This project was developed only for a selection interview task to demonstrate frontend development skills and UI recreation abilities.

## Status

The website UI and interactions are working successfully.
